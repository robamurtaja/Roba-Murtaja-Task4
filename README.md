# Roba-Murtaja-Task4

## Project Description
This project is a fully responsive educational platform built using **HTML, CSS, and JavaScript** 

It allows users to browse available courses, view details for each course, book a course, and track their learning progress locally.

---
## Project Pages

The website includes the following pages:

- **Home Page (index.html)** – Smart recommendations + navigation
- **Courses Page** – Filtering by Category, Level, Format, and Popularity
- **Course Details Page**
- **Book Course Page**
- **My Path Page** – Progress tracking using LocalStorage
- **Contact Page**
- **About Page**
- **Login & Signup Pages** 
- **Splash Screen (Mobile)**
- **Onboarding Screen (Mobile only – first visit)**

The website is fully **responsive** and works seamlessly on desktop, tablet, and mobile devices.

## Key Features

| Feature                  | Description |
|--------------------------|-------------|
| Responsive Design        | Works perfectly on desktop, tablet, and mobile |
| Splash + Onboarding      | Shows only on mobile and only on the first visit |
| Course Filtering         | By category, level, format, and sort |
| Dynamic Course Rendering | Courses loaded from JavaScript array |
| Course Details Page      | Loads details based on ID in URL |
| Booking System           | Stored in LocalStorage for each user |
| My Path Page             | Shows progress and booked courses |
| Mobile Navigation        | Hamburger menu with animated toggle |
